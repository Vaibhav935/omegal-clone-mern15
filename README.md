# OmeGal - Project Overview

## Project Description

OmeGal is a full-stack web application built with React and Node.js. This project demonstrates a modern architecture with a separate frontend and backend serving different responsibilities.

## Project Structure

```
omegal/
├── frontend/          # React + Vite frontend application
│   ├── src/          # Source files
│   ├── public/       # Static assets
│   └── package.json
├── backend/          # Node.js backend server
│   ├── server.js
│   └── package.json
└── README.md         # Project documentation
```

## Getting Started

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Backend Setup
```bash
cd backend
npm install
npm start
```

## Tech Stack

- **Frontend:** React, Vite, ESLint
- **Backend:** Node.js
- **Package Manager:** npm

## Features

- Modern React development with Vite
- Fast Refresh with HMR
- ESLint configuration for code quality
